# 墨阅 (MarkdownViewer)

> **墨色生香，阅见不凡。**
> 
> 一款为 macOS 精心打造的轻量级原生 Markdown 编辑与实时预览器。

## ✨ 特性

- **极速体验**：基于 SwiftUI 原生开发，启动即用，丝滑滚动。
- **纯预览模式**：默认全屏预览，切换编辑后支持左右/上下双栏实时预览。
- **高性能内核**：零依赖渲染技术，即便万字长文也能流畅处理。
- **原生美学**：完美适配 macOS 深色模式，提供沉浸式阅读体验。
- **CSS 样式**：支持自定义 CSS 样式表，打造个性化阅读体验。
- **CLI 支持**：命令行直接打开 Markdown 文件预览。

## ⌨️ 快捷键

| 快捷键 | 功能 |
|---|---|
| `⌘ + E` | 切换编辑 / 预览模式 |
| `⌘ + O` | 打开文件 |
| `⌘ + S` | 保存文档 |
| `⌘ + \` | 切换分栏方向（左右 / 上下） |
| `⌘ + B` | 粗体 |
| `⌘ + I` | 斜体 |
| `⌘ + K` | 插入链接 |

## 🚀 使用方式

### GUI 打开

双击任意 `.md` 文件，或从应用菜单 **文件 → 打开** 选择文件。

### 命令行打开

```bash
open -a "墨阅" /path/to/file.md
```

## 🎨 自定义样式

墨阅支持自定义 CSS 样式表：

1. 在 **设置 → 样式** 中指定自定义 CSS 文件路径
2. 或在 Markdown 文件同目录下放置同名 `.css` 文件（如 `README.css`）
